Pomodoro Timer

Created as part of the Free Code Camp Advanced Front-End Design Curriculum, this is a Pomodoro Timer. The pomodoro timer is a productivity tool that posits a user is more productive with short bursts of work, followed by a short break. The default setting is 25 minutes of work and a 5 minute break. The user can alter this using plus or minus buttons.

    Tools Used:
  • Bootstrap
  • CSS
  • HTML
  • Jquery
    User Stories:
  • I can start a 25 minute pomodoro, and the timer will go off once 25 minutes has elapsed.
  • I can reset the clock for my next pomodoro.
  • I can customize the length of each pomodoro.
