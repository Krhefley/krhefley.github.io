PROCESS

A good UX/UI design process is highly iterative. I complete the steps below several times before reaching the desired design.

  • Assess

    I consider this to be the most crucial phase in both web development and UX design. This is when I begin to extrapolate user needs through interviews, persona creation, surveys, introspection and user testing. Through frequent user interaction, I ensure that my assumptions for design and development remain valid.

  • Design

    The fun phase of the project begins. Using sketching, wireframing and prototypes of various fidelities, I begin to ideate and reify a solution. This phase is highly iterative -- I loop back through the assess phase several times. With user-centered design as the driving force this is when the project starts to take shape.

  • Build

    In this phase, the building begins! Using front-end technologies I build and customize the product to match the intended design. Continuing meetings with stakeholders help to keep the project on track and ensure the product is coming along as envisioned. Intense developer testing is followed by user testing to verify usability.
